# NodeJs com node parser

## Rodando o projeto

Para rodar o projeto no setup de desenvolvimento, devemos utilizar:

```bash
yarn start:dev
```

Para produção:

```bash
yarn prod
```

## `.env`

O `.env` do projeto devera ter as seguintes informações:

```bash
APP_ID=
MASTER_KEY=
APP_NAME=
SERVER_URL=
DATABASE=
JWT_SECRET=
```
